# DateAnalysis

本地数据分析与图表展示桌面软件。

## 当前版本能力
### V1.0 / V1.1
- 本地导入 CSV / XLSX
- 原始数据预览（前1000行）
- 统计结果：最大值、最小值、平均值、中位数、求和、方差、标准差、极差、缺失值
- 多 Y 轴折线对比
- 每条折线颜色自定义
- 平均值参考线开关
- 统计结果导出 CSV / XLSX
- 图表导出 PNG

### V1.2 新增
- 单独的数据处理模块：按列条件删除整行或替换为列均值
- 时间粒度聚合：分钟 / 小时 / 班次（早8晚8两班制） / 天 / 周
- 多文件导入与切换展示
- 多文件按时间合并展示，合并结果仅内存临时保存
- 临时储存区：处理/合并结果生成临时数据集，支持切换、导出、删除
- 原始导入数据标记为"原始"，不被覆盖

## 环境与运行
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app\main.py
```

## 使用说明
1. 导入一个或多个 CSV/XLSX 文件
2. 在左侧“数据集管理（临时储存区）”切换当前数据集，或选中多个文件点“合并选中”
3. 在“数据处理模块”添加条件规则，执行后会生成临时数据集
4. 选择统计列、X 轴、Y 轴，按需设置时间粒度、平均值线、数据点显示
5. 点击“开始分析”查看统计表和折线图
6. 需要落地保存时，导出统计结果、图表或选中的数据集

## 统计与处理说明
- 方差/标准差默认样本统计（ddof=1）
- 时间聚合时 Y 值取窗口内平均值
- 班次规则：08:00-20:00 为早班，20:00-次日08:00 为晚班，晚班归属开始日20:00
