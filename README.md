# DateAnalysis

本地数据分析与图表展示桌面软件，支持 CSV/XLSX 导入、多文件管理、条件数据处理、时间粒度聚合、多 Y 轴折线图、常用统计分析、结果与图表导出。

## 功能特性

- 本地导入 CSV / XLSX 表格
- 多个文件导入与切换展示
- 多文件按时间列合并，自动按时间排序
- 临时储存区：原始数据 / 处理结果 / 合并结果分别管理
- 原始数据表预览
- 自动识别数值列、日期列
- 选择统计列并计算：
  - 最大值
  - 最小值
  - 平均值
  - 中位数
  - 求和
  - 方差
  - 标准差
  - 极差
  - 缺失值数量
- 独立数据处理模块：
  - 按列条件处理
  - 小于 / 小于等于 / 大于 / 大于等于 / 等于 / 不等于 / 为空 / 非空
  - 删除整行
  - 替换为该列均值
- 折线图配置：
  - 单 X 轴
  - 多 Y 轴对比
  - 每条 Y 轴折线可自定义颜色
  - 平均值参考线开关
  - 数据点显示开关
- 时间粒度聚合展示：
  - 原始
  - 分钟
  - 小时
  - 班次（早班 08:00-20:00 / 晚班 20:00-次日08:00）
  - 天
  - 周
- 导出能力：
  - 统计结果导出 CSV / XLSX
  - 当前图表导出 PNG
  - 数据集导出 CSV / XLSX
- 持久化运行日志与错误日志：
  - 日志目录：`logs/`
  - 错误带 traceback，方便定位问题

## 快速开始

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app\main.py
```

## 使用流程

1. 点击“导入文件”或“导入文件夹”加载本地表格
2. 在左侧数据集管理区切换当前数据集，或选中多个文件进行合并
3. 在“数据处理模块”中添加条件规则，执行后会生成临时数据集
4. 选择统计列
5. 配置折线图：
   - X 轴列
   - Y 轴列（支持多选）
   - 颜色
   - 是否显示均值线
   - 是否显示数据点
   - 时间粒度
6. 点击“开始分析”查看统计结果和折线图
7. 需要保存结果时导出统计结果、图表或数据集

## 说明

- 数据表默认预览前 1000 行，但统计和绘图基于当前激活数据集的全量有效数据。
- 方差 / 标准差默认使用样本统计口径（ddof=1）。
- 班次规则：
  - 早班：08:00-20:00
  - 晚班：20:00-次日08:00，归属开始日 20:00
- 临时储存区中的处理/合并结果默认只在本次运行中存在，可通过导出保存。
