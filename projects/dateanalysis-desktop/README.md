# DateAnalysis

本地数据分析与图表展示桌面软件，支持 CSV/XLSX 导入、多文件管理、条件数据处理、时间粒度聚合、多 Y 轴折线图、常用统计分析、结果与图表导出。

## 功能特性

- 本地导入 CSV / XLSX 表格
- 多个文件导入与切换展示
- 机头 / 机尾两类文件分类导入，类别独立的单像素精度 factor（QSettings 记忆）
- 多文件按时间列合并，自动按时间排序；同类别（机头/机尾）内部 concat+排序
- 跨类合同图：机头+机尾按 `时间` outer join 对齐，非时间列加 `[机头]`/`[机尾]` 前缀，单侧缺失保持 NaN 独立不填充
- 同图多 Y 折线对比：可同时勾选 [机头]列 + [机尾]列，颜色/均值线独立控制
- 临时储存区：原始数据 / 处理结果 / 合并结果（含 `机头_合并`/`机尾_合并`/`机头+机尾_跨类合并`）分别管理
- 原始数据表预览
- 自动识别数值列、日期列
- 选择统计列并计算：
  - 最大值
  - 最小值
  - 平均值
  - 中位数
  - 求和
  - 方差
  - 标准差
  - 极差
  - 缺失值数量
- 独立数据处理模块：
  - 按列条件处理
  - 小于 / 小于等于 / 大于 / 大于等于 / 等于 / 不等于 / 为空 / 非空
  - 删除整行
  - 替换为该列均值
  - 缩放数值为mm（支持单列/全部数值列、按类别批量缩放“所有机头/所有机尾”、自动重命名列加(mm)后缀、跳过文本/日期/布尔、支持排除列，float32 单精度语义）
- 折线图配置：
  - 单 X 轴
  - 多 Y 轴对比
  - 每条 Y 轴折线可自定义颜色
  - 平均值参考线开关
  - 数据点显示开关
- 时间粒度聚合展示：
  - 原始
  - 分钟
  - 小时
  - 班次（早班 08:00-20:00 / 晚班 20:00-次日08:00）
  - 天
  - 周
- 导出能力：
  - 统计结果导出 CSV / XLSX
  - 当前图表导出 PNG
  - 数据集导出 CSV / XLSX
- 描述统计分析（基础层）：
  - 综合统计扩展：缺失率、变异系数 CV、偏度、峰度、分位数（P1/P5/Q1/中位数/Q3/P95/P99）、IQR
  - 直方图 + KDE 密度曲线、均值/中位数参考线
  - 多列并排箱线图 + 离群点
  - Q-Q 图（正态性直观诊断）
  - 相关矩阵热力图（Pearson / Spearman）
  - 散点图矩阵
  - 多种分析模式切换：折线趋势 / 描述统计 / 时序监控（Phase 2）
- 持久化运行日志与错误日志：
  - 日志目录：`logs/`
  - 错误带 traceback，方便定位问题

## 使用流程

1. 点击“导入文件”或“导入文件夹”加载本地表格（未分类路径，兼容旧流程）
2. 跨类合同图流程：
   - 点击“导入机头文件”，选择机头表格，输入机头单像素→mm 换算因子（factor）
   - 点击“导入机尾文件”，选择机尾表格，输入机尾单像素→mm 换算因子（factor）
   - （可选）在“数据处理模块”中将“作用范围”选为“所有机头”或“所有机尾”，执行“缩放数值为mm”按类别批量缩放（列名自动加 `(mm)` 后缀）
   - 点击“跨类合同图”生成 `机头+机尾_跨类合并` 数据集
   - 在 Y 列选择面板同时勾选至少一个 `[机头]` 列和一个 `[机尾]` 列
   - 切换时间粒度（原始/分钟/小时/班次/天/周），同图对比两类数据折线
3. 在左侧数据集管理区切换当前数据集，或选中多个文件进行同类别/旧路径合并
4. 切换“分析模式”：
   - 折线趋势：多Y对比/时间粒度聚合/均值线（跨类合同图数据集直接支持）
   - 描述统计：一键生成综合统计/分位数/缺失/箱线/相关矩阵与分布图表（跨类合同图对 `[机头]…(mm)`/`[机尾]…(mm)` 前缀列生效）
   - 时序监控：开发中（Phase 2：SPC/EWMA/Cpk/ACF）
4. 在“数据处理模块”中添加条件规则，执行后会生成临时数据集
5. 选择统计列
6. 配置折线图：
   - X 轴列
   - Y 轴列（支持多选）
   - 颜色
   - 是否显示均值线
   - 是否显示数据点
   - 时间粒度
7. 点击“开始分析”查看统计结果和折线图
8. 需要保存结果时导出统计结果、图表或数据集

## 说明

- 数据表默认预览前 1000 行，但统计和绘图基于当前激活数据集的全量有效数据。
- 方差 / 标准差默认使用样本统计口径（ddof=1）。
- 班次规则：
  - 早班：08:00-20:00
  - 晚班：20:00-次日08:00，归属开始日 20:00
- 临时储存区中的处理/合并结果默认只在本次运行中存在，可通过导出保存。
