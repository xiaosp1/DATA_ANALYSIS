# 数据分析学习日志：分布形态分析三图表解读

> 学习日期：2026-07-03
> 章节：1.2 分布形态分析
> 内容：直方图+KDE、箱线图、Q-Q图 三张基础图表通俗解读

---

## 前言

这三张图是描述性统计分析的"入门三件套"，相当于给一堆数据"拍X光片"，从三个不同角度观察数据的"长相"，是所有后续分析（SPC控制图、过程能力分析等）的基础。

- 直方图+KDE：看**整体形状**（数据扎堆在哪、有没有"山头"）
- 箱线图：看**位置范围和异常值**（大部分集中在哪、谁出格了）
- Q-Q图：看**正态性**（能不能用后续的常规统计方法）

---

## 一、直方图 + KDE密度曲线

### 这张图是什么？

就像统计数据里的"身高分布图"，告诉你"哪个长度的手套最多"。

### 图表构成

| 元素 | 含义 |
|------|------|
| 蓝色柱子（直方图） | 把卷边长度按区间分段，数每个区间里有多少只手套。柱子越高，这个长度的手套越多。 |
| 橙色曲线（KDE） | 把柱子的"尖顶"连起来做平滑处理，相当于给你一个顺滑的轮廓，不用一根根数柱子就能看出哪高哪低。 |

### 怎么看这张图？

1. **看有几个"山头"（峰）**
   - 单峰：只有一个山包，说明数据都集中在一个中心附近，正常
   - 双峰/多峰：有两个或多个山包，**这是危险信号**！说明很可能混了两种或多种不同工艺状态的数据（比如预脱辊间隙偏大/偏小两档、左右模差异、两个班次混在一起），也就是文档里说的"混群"问题

2. **看对称不对称**
   - 对称：山包左右坡度差不多，左右两边"拖尾巴"长度差不多
   - 右偏（正偏态）：山包靠左，右边拖个长尾巴，说明有少数特别长的异常值
   - 左偏（负偏态）：山包靠右，左边拖个长尾巴，说明有少数特别短的异常值

3. **看"胖瘦"**
   - 山包越窄（瘦高）：数据越集中，波动越小，工艺越稳定
   - 山包越宽（矮胖）：数据越分散，波动越大，工艺越不稳定

### 本次数据结论

- 是**单峰分布**，只有一个中心，没有混群问题
- 分布**基本对称**，左右坡度差不多
- 中心位置大概在8.3-8.4附近
- 整体峰形正常，数据集中度尚可

## 二、箱线图（Box Plot）

### 这张图是什么？

专门用来快速看数据的分布范围和抓异常值的，相当于"体检表里的正常值范围 + 异常指标标记"，是工程师最喜欢用的快速排查工具。

### 图表构成（从下往上五个关键位置）

```
        ○  ← 离群点（异常值）
    -------  ← 上须：正常值范围内的最大值
      |     |
      |-----|  ← Q3：75%分位数（箱子顶边）
      |--●--|  ← 中位数：一半比它短一半比它长（"中等水平"）
      |-----|  ← Q1：25%分位数（箱子底边）
      |     |
    -------  ← 下须：正常值范围内的最小值
        ○  ← 离群点（异常值）
```

**补充概念**：
- **IQR（四分位距）** = Q3 - Q1，就是箱子的高度。它代表了**中间50%数据**的波动范围，这部分数据最有代表性，不受极端值影响。
- **离群点判定规则**：小于 Q1 - 1.5×IQR，或者大于 Q3 + 1.5×IQR 的点，就被判定为异常值，画成单独的小圆圈。

### 怎么看这张图？

1. **看中位数位置**
   - 中位数在箱子正中间：数据对称
   - 中位数靠近箱子上边：数据左偏（短的那头尾巴长）
   - 中位数靠近箱子下边：数据右偏（长的那头尾巴长）

2. **看箱子高度（IQR）**
   - 箱子越窄：中间大部分数据越集中，整体波动越小
   - 箱子越宽：中间大部分数据越分散，整体波动越大

3. **看上下胡须长度**
   - 哪根胡须长，说明哪一侧的极端值更多/更远

4. **看离群点（最关键！）**
   - 上下有没有单独的小圆圈？有几个？离箱子远不远？
   - 离群点就是**明显出格的异常样本**，是你重点排查的对象
   - 上面有圈：有异常偏长的卷边
   - 下面有圈：有异常偏短的卷边

### 本次数据结论

- 中位数在箱子中间略偏上，整体分布大致对称
- 箱子宽度适中，中间50%数据集中度较好
- **最上方有3-4个离群点**：存在少数几只卷边**异常偏长**的手套，需要重点关注
- 下方没有明显离群点：异常偏短的情况很少

## 三、Q-Q图

### 这张图是什么？

专门用来检验数据**是否近似服从正态分布**的工具。为什么要检验正态性？因为后续的SPC控制图、Cpk过程能力分析、t检验等很多常用统计方法，都要求数据近似正态，结果才可靠。

### 图表构成

| 元素 | 含义 |
|------|------|
| 红色对角线 | "标准参考线"。如果数据完美符合正态分布，所有点都应该老老实实落在这条线上。 |
| 蓝色散点 | 你实际数据的分位数，按从小到大排列后和理论正态分布的分位数一一对应画出来的点。 |

### 怎么看这张图？

1. **看中间部分**：这是核心！中间大部分点如果紧贴着红线走，说明**数据主体是符合正态的**。
2. **看两头（左下角和右上角）**：
   - 两头都贴着红线：完美正态
   - 右上角往上翘：右尾长，有少数特别大的异常值（就是箱线图里的上离群点）
   - 左下角往下弯：左尾长，有少数特别小的异常值
   - 整体弯成S形：严重偏态，不符合正态分布
   - 点弯成"("或")"形：峰度不对，太尖或太平

### 常见偏离形态对照表

| 图形形态 | 含义 |
|----------|------|
| 点几乎都在红线上 | 完美正态 |
| 中间贴合，只有两头稍微偏离一点点 | 近似正态（实际工业数据基本都是这样，完全可以用正态方法） |
| 右上角明显上翘 | 右偏/有偏大异常值 |
| 左下角明显下弯 | 左偏/有偏小异常值 |
| 整体弯成S形 | 严重偏态，需要做数据变换（比如取对数）或换用非参数方法 |

### 本次数据结论

- **中间绝大部分点都紧紧贴着红线**：数据主体非常接近正态分布
- 只有**右上角最末端有几个点稍微往上翘**：对应箱线图里那几个偏长的异常值，只是小尾巴
- 整体偏离程度很小：**可以认为近似正态分布**，后续SPC控制图、能力分析等方法都可以放心使用

---

## 三图联动综合解读口诀

1. 直方图看"形"：几个峰、对不对称、胖还是瘦
2. 箱线图看"界"：中心在哪、正常范围多大、谁是异常值
3. Q-Q图看"性"：符不符合正态，能不能用后面的高级方法

---

## 本次样本综合结论

| 维度 | 结论 |
|------|------|
| 分布形态 | 单峰分布，基本对称，**没有混群问题** |
| 中心位置 | 卷边长度大部分集中在 **8.3~8.4** 附近 |
| 波动情况 | 中间50%数据集中度尚可，整体波动正常 |
| 异常情况 | 存在**少数几只卷边异常偏长**的样本，需单独排查原因 |
| 正态性 | **近似正态分布**，后续常规统计方法均可使用 |

## 下一步该看什么？

看懂这三张图，只是说你知道了"数据长什么样"。接下来要解决的问题是：
- 这些异常偏长的卷边是什么时候出现的？对应哪个班次/哪个模座？
- 数据随时间是稳定的，还是在慢慢漂移（辊子磨损）？
- 我们的工艺能不能稳定满足规格要求？（过程能力Cpk）

这就是文档后面要讲的：**时序分析、SPC控制图、过程能力分析**。
