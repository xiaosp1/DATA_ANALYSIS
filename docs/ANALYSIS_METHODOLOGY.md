# 丁腈手套卷边长度检测 数据分析方法论

## 0. 业务背景与数据字段

### 0.1 工艺链路
预脱辊（舒展卷边）-> 工业相机视觉检测 -> 脱模机（脱取手套）。
预脱辊的舒展效果直接决定卷边长度，卷边长度过短会导致脱模失败、破洞、拉伸等质量问题。

### 0.2 已知数据字段
- 时间戳：每只手套（或每个手模）经过相机的时刻
- 中指尖距离：腕部卷边到手模中指尖的距离
- 拇指尖距离：腕部卷边到手模拇指尖的距离
- 虎口距离：腕部卷边到手模虎口位置的距离
- 可扩展字段（建议补充）：手模编号/模座号、班次、生产线号、操作员、脱模结果（成功/失败/破损）、预脱辊参数（压力/转速/间隙）

### 0.3 分析总目标
从三个距离指标中挖掘：
1. 当前卷边舒展质量的整体水平与稳定性
2. 卷边长度的时间变化规律与异常模式
3. 卷边长度与脱模质量之间的关联
4. 影响卷边长度波动的关键因素
5. 可操作的工艺优化与预警阈值

## 一、描述性统计分析（基础层）

### 1.1 单变量统计
对中指尖、拇指尖、虎口三个距离列分别计算：

| 统计量 | 业务含义 |
|--------|----------|
| 均值 Mean | 卷边舒展的平均水平，反映预脱辊整体舒展能力 |
| 中位数 Median | 剔除极端值后的中心水平，比均值更稳健 |
| 标准差 Std / 方差 Var | 卷边长度的波动幅度，反映工艺稳定性 |
| 极差 Range（Max-Min） | 波动范围的直观指标 |
| 变异系数 CV = Std/Mean | 消除量纲后的相对波动，便于跨指标/跨产线对比 |
| 偏度 Skewness | 分布对称性：正偏表示存在偏小（舒展不足）的长尾，负偏反之 |
| 峰度 Kurtosis | 分布陡峭程度：高峰度说明极端异常值偏多 |
| 缺失率 | 视觉检测漏检率，反映相机/算法稳定性 |

**效果**：建立卷边长度的基准画像，快速判断"整体是否合格、波动是否正常"。

### 1.2 分布形态分析
- **直方图 + KDE密度曲线**：观察卷边长度是单峰、双峰还是多峰分布。双峰常暗示存在两种工艺状态（如预脱辊间隙偏大/偏小两档、左右模差异）。
- **箱线图（Box Plot）**：直观展示四分位距（IQR）与离群点，识别过短/过长卷边。
- **Q-Q 图**：检验数据是否近似正态分布，为后续能力分析和控制图选方法提供依据。

**效果**：识别分布异常形态，发现"混群"问题（不同班次/机台/模座混在一起）。

### 1.3 三指标相关性
- **Pearson 相关系数矩阵 + 热力图**：看中指尖、拇指尖、虎口三个距离是否同步变化。
- **散点图矩阵（Pair Plot）**：两两指标的散点，观察是否线性、是否存在聚类。
- 若三指标高度相关（r > 0.9），后续可用主成分降维或选最具代表性的单指标监控。

**效果**：明确三个测点之间的一致性，避免冗余监控或漏监控。

## 二、时序分析与稳定性（监控层）

卷边长度是典型的时序数据，按手模顺序/时间顺序排列，必须从时间维度观察变化规律。

### 2.1 趋势分析
- **折线时序图**：以时间戳/手模序号为X轴，距离为Y轴，直观观察整体上升/下降/漂移趋势。
- **滑动均值（Moving Average）**：窗口N=50/100/500只手套的滑动均值，平滑掉单只波动，暴露慢漂移。
- **滑动标准差**：窗口内标准差时序，观察波动本身是否在变大（预示设备异常老化）。
- **线性回归斜率**：对一段时间窗口拟合 y = a*t + b，斜率a显著不为0则存在系统性漂移（如预脱辊磨损、温度变化）。

**效果**：发现预脱辊磨损、胶料批次变化、温湿度变化等慢变量引起的趋势漂移。

### 2.2 周期性分析
- **班次对比箱线图**：早班/晚班箱线图并排，判断班次差异。
- **按小时聚合均值/方差**：识别开机预热阶段（前1-2小时）与稳定运行阶段的差异。
- **自相关图（ACF）**：检测序列是否存在周期性（如手模转盘周期、模座循环周期）。
- **模座/手模编号分组箱线图**：若有手模ID，可直接定位到是个别模座问题还是普遍性问题。

**效果**：区分"全时段均匀问题"和"特定班次/特定模座/开机阶段"问题，精准定位根因。

### 2.3 统计过程控制 SPC（核心工业方法）
SPC 是制造业最成熟的过程稳定性监控方法，推荐作为核心方法论。

#### 2.3.1 控制图选型
- **X-bar R 图（均值-极差图）**：按合理子组（如每50只/每组模座/每分钟）分组，监控组均值与组内极差，最适合连续生产过程。
- **I-MR 图（单值-移动极差图）**：若采样频率低或子组大小为1，用单值图。
- **EWMA 图（指数加权移动平均）**：对小偏移敏感，适合检测预脱辊缓慢磨损这类渐变异常。
- **CUSUM 图（累积和图）**：对过程均值的小持续偏移敏感，适合检测工艺漂移。

#### 2.3.2 控制限计算
- UCL/LCL = 均值 ± 3*sigma（基于稳态历史数据估计）
- 超出控制限 = 特殊原因变异，需要排查
- 控制限内的连续趋势（如连续7点上升/下降、连续9点在中心线同侧）也判异（Western Electric规则/Nelson规则）。

**效果**：区分"正常随机波动（普通原因）"和"异常事件（特殊原因）"，实现科学报警，避免盲目调机。

### 2.4 过程能力分析 Cp/Cpk
在过程受控（统计稳态）前提下：
- **Cp = (USL - LSL) / 6*sigma**：衡量过程固有波动是否满足规格宽度。
- **Cpk = min((USL-μ)/3σ, (μ-LSL)/3σ)**：衡量中心是否偏移，Cpk >= 1.33 为一般合格，>=1.67 为优秀。
- 需要先与工艺工程师共同确定卷边长度的规格上下限（USL/LSL）。

**效果**：量化当前预脱辊+视觉系统"能做到多好"，为设备验收、参数优化提供量化指标。

## 三、异常检测与根因定位（诊断层）

### 3.1 离群点检测
- **IQR法**：超出 [Q1-1.5*IQR, Q3+1.5*IQR] 的点标记为离群点，简单稳健，适合初筛。
- **Z-Score法**：|z| > 3 标记为异常，要求数据近似正态。
- **MAD法（中位数绝对偏差）**：以中位数为中心，对极端值更稳健，适合污染数据。
- **孤立森林（Isolation Forest）**：多维度异常检测（同时看三个距离+时间间隔等），适合发现"单点异常"。
- **DBSCAN聚类**：将低密度区域点标为异常，能发现小簇异常模式。

**效果**：自动标记卷边长度异常手套，联动后续脱模结果做验证。

### 3.2 异常模式分类
将时序异常分为若干典型模式，对应不同根因：

| 异常模式 | 图形特征 | 可能根因 |
|----------|----------|----------|
| 孤立尖刺 | 单点突然极大/极小 | 视觉误检、胶料瑕疵、手模异物 |
| 阶跃偏移 | 均值突然跳变 | 换班、调机、换胶料批次、预脱辊参数变更 |
| 渐变漂移 | 均值缓慢上升/下降 | 预脱辊磨损、温度变化、辊面沾附 |
| 周期性波动 | 按模座/转盘周期波动 | 个别模座磨损、转盘偏心、辊子不平衡 |
| 方差异常 | 均值正常但波动突然变大 | 设备松动、气压不稳、原料不均匀 |

**效果**：把"异常"从笼统的好坏分类细化到可指导维修的模式层面。

### 3.3 分段变点检测（Change Point Detection）
- **PELT 算法**：自动找出时序中均值/方差发生阶跃的时刻点。
- **滑动窗口t检验**：比较相邻两段窗口均值是否显著差异。
- **Bayesian 在线变点检测**：实时检测变点，适合在线监控场景。

**效果**：自动定位"工艺从什么时候开始变了"，与换班、调机、换料记录做时间对齐，快速定位根因。

### 3.4 多变量协同分析
- 三个距离同时异常 vs. 单一距离异常：
  - 三个都异常：大概率是预脱辊整体问题（压力、间隙、转速）。
  - 只有拇指尖异常：可能是拇指位置预脱接触不良，或该方向辊面问题。
  - 只有虎口异常：虎口位置舒展不充分，可能与预脱辊对虎口位置的包覆角度有关。
- **马氏距离（Mahalanobis Distance）**：多变量异常得分，考虑指标间相关性，比单变量阈值更灵敏。

**效果**：从"单个指标异常"升级为"异常模式定位"，指导机械调整的具体方向。

## 四、关联与因果分析（建模层）

### 4.1 卷边长度 vs 脱模质量
这是最核心的业务问题：**卷边长度要多长才够？**

- **分组对比**：按脱模结果（成功/失败/破损/粘连）分组，绘制卷边长度的箱线图/小提琴图，直观对比分布差异。
- **假设检验**：
  - t检验/Mann-Whitney U检验：失败组与成功组卷边长度是否显著差异。
  - ANOVA：多种失败模式之间是否有差异。
- **逻辑回归（Logistic Regression）**：以脱模成功/失败为因变量，三个距离为自变量，拟合概率模型 p(success) = f(d1, d2, d3)。
  - 输出各指标的Odds Ratio，识别哪个距离对脱模成功率影响最大。
  - 得到**临界阈值**：如中指尖距离 < X mm时，失败率从2%跳升到15%。
- **ROC曲线**：选定阈值后评估预测能力，AUC衡量区分度。

**效果**：从"卷边长度有波动"上升到"波动是否影响脱模成功率、临界值在哪里"，给出可量化的工艺规格建议。

### 4.2 与工艺参数的关联
若能采集预脱辊参数（压力P、转速V、间隙G）：
- **散点图 + 相关性**：各参数与卷边长度的Pearson/Spearman相关系数。
- **多元线性回归**：L = a0 + a1*P + a2*V + a3*G + e，量化各参数影响方向和幅度。
- **响应面分析（RSM）**：参数交互作用分析，寻找最优参数组合。
- **偏最小二乘 PLS**：参数多且共线性时用。

**效果**：找到预脱辊的最优工艺窗口，指导参数调优方向。

### 4.3 与环境/班组因素的关联
- **班次/操作员**：箱线图对比，ANOVA检验显著性，判断是否有人为因素。
- **温湿度**：若有车间环境数据，做相关性与滞后相关分析（温度变化可能滞后影响胶料性能）。
- **原料批次**：按批次分组箱线图，识别批次差异。
- **开机时长（暖机效应）**：按开机后N分钟切片，观察卷边长度随暖机的变化曲线。

**效果**：识别管理因素、环境因素对质量的贡献，区分"设备问题"和"人/料/法/环问题"。

### 4.4 空间/结构关联
- **左右模对比**：若数据有左右模标记，配对t检验，判断是否有系统性左右偏差。
- **模座热力图**：按模座编号排列的均值/方差热力图，快速定位"问题模座"。
- **沿转盘周向分布**：观察卷边长度是否在转盘某个角度位置系统性偏差。

**效果**：定位机械结构问题（如辊子平行度、转盘偏心、喷酸/喷淋不均）。

## 五、高级建模与预测（优化层）

### 5.1 质量预测与提前预警
- **二分类模型**：以卷边长度（+其他特征）预测脱模失败风险。
  - 模型可选：逻辑回归、随机森林、XGBoost、SVM。
  - 评估：准确率、召回率、精确率、F1、AUC，重点关注召回率（漏检失败的代价）。
- **时序预测**：用ARIMA/Prophet/LSTM预测未来N分钟的卷边长度均值，若预测将低于阈值则提前报警。
- **在线评分**：每只手套计算风险分数，超过阈值实时通知操作员干预。

**效果**：从"事后分析"升级为"事前预警"，降低不良率。

### 5.2 根因挖掘
- **决策树/随机森林特征重要性**：把候选因素（时间、班次、模座、参数、环境）都作为特征，预测卷边长度或脱模结果，输出特征重要性排序。
- **SHAP值分析**：解释每个因素对单个预测的贡献方向和大小，黑盒模型白盒化。
- **关联规则挖掘（Apriori）**：挖掘"卷边短 + 拇指异常 + 某班次"这类频繁共现模式。

**效果**：从众多可能因素中定量排序，避免经验主义的调机方式。

### 5.3 工艺窗口优化
- **工艺包络（Process Window）**：以三个距离为维度，画出脱模成功率高的区域边界，形成合格"体积"。
- **贝叶斯优化**：在预脱辊参数空间中智能搜索最优组合，比全因子DOE更省试验次数。
- **SPC + 规格的双界管理**：同时维持控制限（稳定性）与规格限（合格性），过程既要"稳"也要"准"。

**效果**：给出预脱辊参数的最优设定范围，实现质量最优、波动最小。

### 5.4 聚类与模式识别
- **K-Means / GMM 聚类**：对手套按三距离向量聚类，发现"质量分型"（如：正常类、卷边偏短类、拇指舒展不足类）。
- **时间序列聚类（K-Shape / DTW）**：对一段时间窗口的曲线形状聚类，识别典型异常曲线族。
- **自编码器（Autoencoder）**：正常数据训练重建模型，重建误差大的即异常，适合无标签场景。

**效果**：在无标签或标签不完整时，自动发现隐藏的质量模式和未知异常。

## 六、分析落地路线图建议

基于当前已有数据（时间戳 + 三个距离），建议分四阶段推进：

### 阶段一（立即可做，数据已具备）
1. 描述性统计：三指标均值/标准差/CV/分位数/缺失率
2. 分布可视化：直方图、箱线图、时序折线图
3. 三指标相关性分析
4. 班次/小时聚合对比（需有班次字段，可从时间戳推导）
5. I-MR 单值控制图，初建控制限
6. IQR离群点检测，标记异常手套

### 阶段二（建议补充数据采集）
1. 增加字段：模座/手模编号、脱模结果标签、预脱辊参数（压力/转速/间隙）
2. 模座分组分析，定位问题模座
3. 卷边长度 vs 脱模结果分组对比，做t检验与逻辑回归，得出临界阈值
4. X-bar R 控制图替代 I-MR（按模座/时间分组）
5. Cpk 过程能力分析（在阈值确定后）
6. 变点检测（PELT），自动识别工艺变化时间点

### 阶段三（数据积累后）
1. 多元异常检测（马氏距离、孤立森林）
2. 脱模失败预测模型（随机森林/XGBoost），在线实时评分
3. 工艺参数多元回归/响应面，指导参数调优
4. SHAP根因分析
5. EWMA/CUSUM 精细控制图，检测缓慢漂移

### 阶段四（闭环控制）
1. 时序预测（Prophet/LSTM），提前预警漂移
2. 贝叶斯优化工艺参数
3. 预脱辊参数闭环推荐系统
4. 数字孪生与仿真

| 11 | Z-Score / MAD / 孤立森林 异常检测 | 诊断层 | 三距离(+时间) | 自动标记异常样本 |
| 12 | 变点检测（PELT / 贝叶斯） | 诊断层 | + 时间戳 | 自动定位工艺跳变时间点 |
| 13 | 马氏距离多变量异常得分 | 诊断层 | 三距离 | 综合三指标的多维异常检测 |
| 14 | 异常模式分类（尖刺/阶跃/漂移/周期/方差异常） | 诊断层 | + 时间戳 | 把异常对应到根因类型 |
| 15 | 分组对比 + t检验/ANOVA/Mann-Whitney | 建模层 | + 脱模结果/班次/批次 | 验证因素对卷边长度的显著性影响 |
| 16 | 逻辑回归 + ROC 临界阈值 | 建模层 | + 脱模结果 | 确定卷边长度临界值，量化失败风险 |
| 17 | 多元线性回归 / 响应面 RSM | 建模层 | + 预脱辊参数 | 量化参数对卷边长度的影响方向和幅度 |
| 18 | 随机森林/XGBoost + SHAP | 建模层 | 多因素 | 特征重要性排序，根因白盒化 |
| 19 | 聚类（K-Means/GMM/K-Shape） | 优化层 | 三距离(+时间) | 发现无标签下的隐藏质量分型 |
| 20 | 时序预测（ARIMA/Prophet/LSTM） | 优化层 | 长时间序列 | 提前预警未来漂移趋势 |
| 21 | 贝叶斯优化 | 优化层 | 参数试验数据 | 寻找最优预脱辊参数组合 |
| 22 | 自编码器无监督异常检测 | 优化层 | 大量正常样本 | 无标签下识别未知异常模式 |

## 八、关键建议

1. **数据补全优先**：强烈建议增加"脱模结果标签"和"模座/手模编号"字段，这两个字段是后续所有根因分析和阈值制定的关键钥匙。没有脱模结果，只能做稳定性分析，无法回答"到底多短才出问题"。
2. **SPC先行**：在数据量足够（建议先积累2-3天稳态数据）后立即建立I-MR/X-bar R控制图，这是制造业最实用、投入产出比最高的方法。
3. **分层次推进**：不要一上来就上机器学习。先把描述统计、SPC、异常检测、分组对比做扎实，往往80%的问题在这一阶段就能定位。
4. **可视化优先**：所有分析都应配合图形（折线图、箱线图、直方图、热力图），工艺工程师看图比看数字更能发现问题。
5. **闭环思维**：分析最终要服务于两个动作：一是"报警"（实时告诉操作员现在有问题），二是"改进"（告诉工程师往哪个方向调参数）。所有分析方法都应最终落到这两个动作上。
