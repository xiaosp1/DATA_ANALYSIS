# 脱模工艺概述

> 本文件沉淀脱模优化项目所涉及的制造工艺、关键指标体系与数据分析目标，基于 `app/services/` 领域代码中的实际实现。

---

## 1. 工艺背景

本项目面向 **橡胶/弹性体手套脱模工序** 的数据分析，核心关注点是通过采集机头工艺参数、分析其对机尾最终产品质量的影响，寻找最优工艺窗口。

产品核心质量指标是 **指数-s**（手套脱模后挂的手指数）：
- **指数-s = 4**：完美脱模（无多余挂手）
- **指数-s > 4**：挂手过多，需额外摘手，降低效率
- **指数-s < 4**：挂手不足，可能导致脱模不完整或手套损伤

## 2. 关键指标体系

### 2.1 机头工艺参数（输入特征）

从 CSV/Excel 数据中自动识别的数值列，以 `[机头]` 为前缀（跨类合并后形成宽表）：
- 温度、压力、时间、速度等物理参数
- 数据按类别（如机头/机尾分类）组织，支持多批数据跨类同图对比
- 数值列支持缩放（像素→mm 单位转换），缩放因子通过 `ProcessingRule` 配置

### 2.2 机尾质量指标（输出目标）

以 `[机尾]` 为前缀的列，核心是 **指数-s**：
- 目标理想值：4.0（±0.5 视为近理想）
- 分布统计：均值、标准差、精确等于理想的占比、近理想占比
- 支持多类别（多批次/多线体）数据合并分析

### 2.3 工艺状态列

自动识别或手动选择的分类列，用于分组分析：
- 识别规则：`nunique ≤ 12`、整数类型、关键词匹配（"指数"、"state"、"状态"、"label"、"index"）
- 默认排除状态 0，默认分析所有非 0 状态
- 单组样本不足 30 条时标记为"结论不可靠"

## 3. 数据流架构

```
原始 CSV/Excel
    │
    ├── 编码自动检测（UTF-8-SIG / UTF-8 / GBK / GB18030）
    ├── 列类型自动推断（datetime / numeric / categorial）
    ├── 分类/排除列选择
    ├── 数值缩放（像素→mm，float32 精度，支持 (mm) 后缀）
    ├── 数据处理规则应用（delete_row / replace_mean / scale_by_factor）
    ├── 多数据集分类管理（category / kind: original/processed）
    ├── 时间聚合（原始/分钟/小时/班次/天/周）
    │   └── 班次定义：早班 08:00-20:00，晚班 20:00-次日 08:00
    │
    ├── 时间就近对齐（merge_asof, ±1s tolerance）
    ├── 跨类合并（同批号不同类别数据对齐）
    │
    ▼
  分析引擎
    ├── 描述统计（含偏度、峰度、分位数、IQR 离群点）
    ├── 相关性分析（Pearson + Spearman，含 null 安全）
    ├── 工艺窗口分析（单变量 μ±σ/μ±2σ，多状态分组）
    ├── 规则挖掘（贪心分类树，max_depth=3, F0.5 排序）
    ├── 特征重要性（ANOVA F 值）
    ├── 机头→机尾归因（双相关系数 + 分组统计 + 分箱 + 阈值规则）
    │
    ▼
  AI 解读（可选）
    ├── OpenAI 兼容 API（gpt-4o-mini 等）
    ├── DeepSeek 兼容 API（deepseek-chat 等）
    ├── 仅传聚合统计，不传原始行数据（符合数据红线）
    ├── 超时 5~300s，可热配置
```

## 4. 跨类合并与机头/机尾同图

V1.7 引入的核心功能：
- 同批号下机头数据与机尾数据通过 **时间就近匹配**（`merge_asof`，nearest 模式，±1s）对齐
- 对齐后形成宽表，机头特征列加 `[机头]` 前缀，机尾列加 `[机尾]` 前缀
- 支持多类别数据同时显示在同一图表上（按分类列分组绘制）
- 支持小多图（small multiples）模式：每张图展示一个类别的轨迹

## 5. 数据分析目标

| 目标 | 方法 | 输出 |
|------|------|------|
| 质量现状评估 | 描述统计 + SPC 控制图 | 均值/标准差/分布形态/控制限 |
| 参数相关性 | 散点矩阵 + 相关性矩阵 | Pearson/Spearman 相关系数 |
| 工艺窗口 | 单变量分状态窗口分析 | 各状态下 μ±σ/μ±2σ 推荐范围 |
| 归因分析 | 贪心分类树 + ANOVA F | 关键特征排序 + 判别规则 |
| 机头→机尾传导 | 跨类归因报告 | Top 机头特征对指数-s 的影响量 |
| 工艺优化建议 | AI 解读（聚合统计驱动） | 结构化工艺窗口总结与优化建议 |
